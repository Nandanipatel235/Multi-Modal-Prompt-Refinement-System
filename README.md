# Multi-Modal Prompt Refinement System (MMPRS)

## Overview
Transforms text, images, documents into standardized AI prompts with confidence scoring.

## Features
- Multi-modal input: text + images + documents
- Intent extraction & classification
- Dynamic confidence scoring (0.2-1.0)
- Rejection handling for unclear inputs
- 5+ test cases covering all scenarios
